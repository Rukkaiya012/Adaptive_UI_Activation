# Behavior Signals

## EXCESSIVE_UNDO
Triggered when user performs 3+ undo actions in sequence. Indicates potential confusion or difficulty with the interface.

## REPEATED_HOVER
Triggered when user hovers over elements 3+ times consecutively. Suggests uncertainty about element functionality or poor affordances.

## LONG_DWELL
Triggered when user dwells on an element for >3 seconds. Indicates cognitive load, confusion, or decision-making difficulty.

## FAST_ACTION
Triggered when user completes an action in <500ms. Suggests high confidence, expertise, or potential accidental interactions.

## BACKTRACK
Signal defined but not implemented. Likely intended to track navigation reversals or workflow interruptions.
